\begin{abstract}
Retrieval-augmented generation (RAG) evaluators often reduce a trace to scalar
quality scores, leaving developers to reconstruct which claim failed, which
evidence mattered, and which component should change. We introduce
\textsc{ContextTrace}, a local-first evidence-chain forensics system that maps
answer claims to retrieved spans, citation edges, source conditions,
abstention decisions, failure labels, root causes, and repair suggestions. A
frozen pre-review evaluation covers 150 public-document diagnostic cases, 200
RAGTruth cases, a same-ID RAGAS baseline, and eleven registered ablation rows.
The current system obtains 0.955 failure-label macro-F1 and zero dangerous
false greens on the RAGTruth sample, but claim-verdict macro-F1 is 0.337 and
independent label review remains incomplete. We additionally run a controlled
1,410-judgment LLM-simulated protocol stress test. The simulated evidence-chain
condition improves root-cause accuracy but not mean actionability, so it does
not support the preregistered positive RQ4 claim and does not replace human
validation. We release a deterministic experiment and anonymous-artifact
workflow with fail-closed claim gates.
\end{abstract}
