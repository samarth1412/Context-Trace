# Saved Response Fixtures

Place saved RAG endpoint response JSON files here only when they come from a documented public-source app run or a reproducible local validation run.

Each checked-in response should have a matching note in `validation/public_rag_apps.md` with:

- repository URL and commit
- command used to generate the response
- query
- field mapping
- ContextTrace commands used
- validation result summary

Generated scratch responses should stay outside git.
